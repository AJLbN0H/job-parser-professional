# Job Parser Professional

Профессиональный консольный инструмент для агрегации и управления вакансиями с HeadHunter. Проект выполнен с использованием строгих принципов ООП, обеспечивающих расширяемость и высокую производительность.

## Технологии и архитектура
- Python 3.12+
- API Integration: Работа с HeadHunter API через библиотеку requests.
- ООП Архитектура:
  - Абстракция: Использование ABC для создания интерфейсов API и файловых хранилищ.
  - Оптимизация: Применение __slots__ в классе Vacancy для экономии памяти.
  - Инкапсуляция: Глубокая валидация данных (ID, зарплата, ссылки).
- Data Storage: Работа с локальным JSON-хранилищем через модульную систему классов.
- Testing: Полное покрытие функционала тестами (Pytest) с использованием unittest.mock.

## Основные возможности
1. Поиск вакансий: Интерактивный поиск по ключевым словам напрямую через API HH.ru.
2. Фильтрация и сортировка: Получение Топ-N вакансий по зарплате и сравнение вакансий между собой.
3. Управление хранилищем: Сохранение в JSON, поиск по ID и полная очистка базы.
4. Безопасность: Защита от некорректных данных (пустые зарплаты, битые ссылки).

## Структура проекта
- api.py — Логика сетевых запросов к API.
- vacancies.py — Модель данных вакансии с валидацией.
- file_work.py — Классы для работы с JSON-файлами.
- utils.py — Вспомогательные функции интерфейса.
- main.py — Точка входа в приложение и CLI-меню.

## Тестирование
Для запуска тестов используйте команду: pytest

## Установка и запуск
1. Клонируйте репозиторий.
2. Установите зависимости: pip install -r requirements.txt
3. Запустите приложение: python main.py
